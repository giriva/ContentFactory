# Requirements

## Minimum system requirements
- Linux, macOS, or Windows with Git installed
- Python 3.10 or newer
- `pip` available with Python
- An Anthropic API key for the orchestrator

## Software dependencies
- `anthropic`
- `PyYAML`

## Environment variables
- `ANTHROPIC_API_KEY`: required for the orchestrator to call Claude

## Optional tools
- `ffmpeg` for video assembly workflows
- `git` for repository management
